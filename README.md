# social-media-blog

This project was created to help me learn how to build websites. Includes API access.

## Stack

* Python
* Flask
* HTML
* Bootstrap
* SQLite
* JWT

Created in Atom

# Instructions

* Database setup
```
flask db init
flask db migrate
flask db upgrade
```
* Run app
```
flask run
```
* Access @ http://localhost:5000
* API requires registered user

* Live Demo @ https://social-media-blog-hetfdex.herokuapp.com/
